# Skill Swap App

A React Native mobile application that enables users to exchange skills, connect with learners, and collaborate with others.

## Status

🚧 Work in Progress

## Current Features

* Login Screen
* Signup Screen
* User Profile UI
* Home Screen
* Search Screen
* Messaging UI
* Navigation System
* Redux State Management

## Tech Stack

* React Native
* TypeScript
* Redux Toolkit
* React Navigation
* Firebase (planned)

## Planned Features

* Firebase Authentication
* Email Validation
* Skill Matching
* Real-time Chat
* Session Scheduling
* Push Notifications

## Known Issues

* Logo rendering issue on some devices
* Email validation not yet implemented
* Authentication backend not connected

## Project Goal

To create a platform where users can teach skills they know and learn skills from others through a skill exchange system.
